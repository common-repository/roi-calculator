== Description ==


**For using This plugin you Can Know ratio that compares the gain or loss from an investment relative to its cost.**

**you can know the result of profit or lose derived from any investment and get price of investment gain.**

**Also know calculate annualized rate of return which is equivalent annual return an investor receives over a given period.**

To run this plugin you need to put shortcode in your page:
([roi_calc])
